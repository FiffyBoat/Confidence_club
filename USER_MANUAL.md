# Confidence Club Members - User Manual

## 1) Quick Start (Admin)
1. Open the app URL in your browser.
2. Admin login: `/admin/login`
3. After login, use the sidebar to navigate:
   - Members
   - Dues
   - Contributions
   - Special Contributions
   - Donations
   - Income / Expenses
   - Loans
   - Receipts
   - Reports
   - Admin Settings

Public access is available at `/viewer` (no login required).  
Public member directory: `/viewer/members`.

---

## 2) Roles & Access
**Admin**
- Full access.
- Manages users (admin/treasurer only).
- Controls public visibility via **Admin -> Settings**.

**Treasurer**
- Records members, contributions, dues, incomes, expenses, loans, and receipts.
- Sees reports and operational dashboards.

**Public Visitors (no login)**
- Can see the public dashboard at `/viewer`.
- Public directory at `/viewer/members`.
- Only see sections enabled by admin settings.

---

## 3) Public Dashboard (No Login)
The public dashboard can display (admin-controlled):
- Constitution link
- Announcements
- Upcoming meetings
- Member directory (limited)
- Birthday highlights
- Special contributions list (paid + description)
- Transparency snapshot

**Admin Control:** Use **Admin -> Settings** to toggle what appears.

---

## 4) Members
**Add Member**
- Members -> Add Member
- If admission fee is not ready, **uncheck** "Record admission fee" and save.
- Admission fee is **GHS 200** and can be recorded later.

**Edit Member**
- Members -> Edit
- You can later **check admission fee** and provide payment details.

**Delete Member**
- Deleting a member removes all related data.

---

## 5) Contributions & Dues
**Contributions**
- Contributions -> Add
- Select type (Admission Fee, Monthly Dues, Extra Levies, etc.)

**Dues**
- Dues -> Record payment for Monthly Dues

---

## 6) Special Contributions
- Minimum amount: **GHS 100**
- Purpose/description is required.
- Recorded under **Special Contributions**
- Appears on public dashboard with "Paid" and description (if enabled in settings).

---

## 7) Donations
- Donations track pooled special contributions.
- Use Donations to record distributions from the pool.

---

## 8) Income & Expenses
- Income -> Record all non-membership income.
- Expenses -> Record club expenses and categories.

---

## 9) Loans
- Loans -> Create & track loan status.
- Loan Repayments -> Record repayments from members.

---

## 10) Receipts
Receipts are generated automatically for:
- Contributions
- Special contributions
- Income
- Loan repayments

Use **Receipts** to view, filter, and download.

---

## 11) Reports
Use **Reports** to review:
- Financial summary
- CCM summary exports (PDF/CSV)
- Period-based statements

---

## 12) Announcements & Meetings (Admin)
**Announcements**
- Admin -> Announcements -> Publish updates for the public dashboard

**Meetings**
- Admin -> Meetings -> Schedule upcoming meetings for the public dashboard

---

## 13) Constitution (Admin)
Upload the constitution document via **Admin -> Settings**.  
It becomes visible on the public dashboard if enabled.

---

## 14) Viewer Visibility Controls (Admin)
**Admin -> Settings**
Toggle what appears on the public dashboard:
- Announcements
- Meetings
- Member directory
- Birthday highlights
- Special contributions
- Transparency snapshot
- Constitution

---

## 15) Public Members Directory
URL: `/viewer/members`
- Search by name, membership ID, phone, or email.
- Public contacts show phone + email.
